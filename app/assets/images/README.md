## default-userpic-75.png

Default user avatar placeholder, sourced from the
[freefeed-server](https://github.com/FreeFeed/freefeed-server) project.
The original GIF is animated (its first frame has a slightly gray
background, causing a visible blink on each loop), so we keep a static
PNG extracted from one of its white-background frames.
